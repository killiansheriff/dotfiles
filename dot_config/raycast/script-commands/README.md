# Raycast Script Commands

Quick app switching scripts for Raycast.

## Setup

1. Open Raycast Settings (⌘ + ,)
2. Go to **Extensions** → **Script Commands**
3. Click thge + icon, **Add Script Directories** and add `~/.config/raycast/script-commands`
4. Assign hotkeys to each command:
   - `Open Cursor` → ⌘1
   - `Open Chrome` → ⌘2
   - `Open iTerm` → ⌘3
   - `Open Slack` → ⌘4

## Scripts

| Script | App | Suggested Hotkey |
|--------|-----|------------------|
| open-cursor.sh | Cursor | ⌘1 |
| open-chrome.sh | Google Chrome | ⌘2 |
| open-iterm.sh | iTerm2 | ⌘3 |
| open-slack.sh | Slack | ⌘4 |
